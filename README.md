# ETH Fraud Sentinel

React + TypeScript app (Vite) for Ethereum fraud detection dashboards and tools.

## Scripts

- dev: Start the dev server
- build: Production build
- preview: Preview the production build locally

## Getting started

```sh
npm install
npm run dev
```

## Tech stack

- React 18
- Vite 5
- TypeScript
- Tailwind CSS
- shadcn/ui
